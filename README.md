# SavirCSS
## A free, open-source CSS library.

![SavirCSS](logo.png)

### Add to your website
Add the following code to your head tag:
```
<link rel="stylesheet" href="https://savirsingh.github.io/SavirCSS/main.css">
```

### See all the styles
